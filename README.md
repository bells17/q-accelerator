# q-accelerator


## Features

- 行番号を表示
- コードのコピー機能
- 特定ユーザーの投稿を非表示
- 自動いいね/自動ストック
- 既読記事を非表示
- 閲覧履歴の検索

## Installation
Chrome ウェブストアにアップする予定です。

## Develop

``` sh
npm install
npm run build
```

extensionフォルダが作られます。  
[chrome://extensions/](chrome://extensions/)にアクセス。  
「パッケージ化されていない拡張機能を読み込む」からextensionフォルダを選択します。
